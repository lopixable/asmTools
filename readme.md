# asmTools
This script allow you to compile and link an asm (intel) code with basic options.

# Install for linux (debian based):

## Update rights
    $ sudo chmod +x asm.sh

## Move to commands directory
    $ sudo mv asm.sh /usr/bin/asm

## Usage
    $ asm (-d) nom_programme

## Option
    -d    add debug info to the executable

## Require
    - nasm
    - ld
